# TASK06-SALES-DB
This project creates a small SQLite database (sales_data.db) with a single sales table and runs basic SQL queries using Python.

Steps Done

Created sales_data.db using Python and added sample sales data.

Connected to the database using sqlite3.

Ran a SQL query to calculate total quantity sold and total revenue for each product.

Loaded results into pandas and printed the summary.

Created a simple bar chart using matplotlib.

Tools Used

Python

sqlite3 (built-in)

pandas

matplotlib

Jupyter Notebook
